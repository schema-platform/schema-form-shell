## OA-01 公告列表 `oa-notice-list`

**Page Pattern**：`CrudList`（数据源 **Notice API**，非 submission）  
**开源参照**：O2OA CMS [列表配置](https://www.o2oa.net/cms/train/59.html)

### 区域布局

```
[A] 标题关键词|类型 dict|发布日期|状态(已发布/草稿)
[B] 发布公告(navigate publish)|导出
[C] 标题(link)|类型(tag)|发布人|发布时间|已读率|置顶|状态
[D] 查看→阅读详情 | 编辑(权限) | 下架
```

### 功能点

| # | 功能 | 来源 |
|---|------|------|
| F1 | 已发布公告列表 | O2OA CMS 列表 |
| F2 | 标题进阅读页 | 阅读表单 |
| F3 | 已读率 | O2OA 读者统计 |
| F4 | 类型/状态筛选 | CMS 分类 |
| F5 | 置顶标识 | 发布表单 |

### Editor

E-45 CrudList · listApi: GET `/api/notices` · detail: navigate oa-notice-detail

### 缺口

**S-05 Notice CRUD** 阻塞 L3

## OA-02 公告发布 `oa-notice-publish`

**Page Pattern**：`ContentPublish`（E-49 编辑态）  
**开源参照**：O2OA [发布类编辑表单](https://www.o2oa.net/cms/content/285.html)

### 表单字段

| 字段 | Widget |
|------|--------|
| 标题 | FgInput |
| 类型 | FgSelect dict://notice_type |
| 发布范围 | FgTreeSelect 部门 + FgSwitch 全员 |
| 正文 | FgRichtext |
| 附件 | FgUpload |
| 置顶 | FgSwitch |
| 有效期 | FgDate 范围 |

### 功能点

F1 读者控件(范围) · F2 发布按钮 · F3 可选审批后发布 · F4 草稿 beforeSave

### Editor

E-49 FgContentPublishPage(editSchemas) · publishApi POST
